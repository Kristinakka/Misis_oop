### 2.2 Generic-класс TypedCollection

Класс `TypedCollection[T]` - обобщённая типизированная коллекция, способная хранить элементы только одного типа `T`.

**Основные методы:**
```
        | Метод | Описание | Возвращаемый тип | 
 `add(item: T)`- Добавляет элемент в коллекцию | `None` |
 `remove(item: T)` - Удаляет элемент из коллекции | `None` |
 `get_all()` -  Возвращает копию списка всех элементов | `List[T]` |
 `size()` -  Возвращает количество элементов | `int` |
 `is_empty()` - Проверяет, пуста ли коллекция | `bool` |
 `clear()` - Очищает коллекцию | `None` |
 `contains(item: T)` - Проверяет наличие элемента | `bool` |
 `find(predicate)`- Находит первый элемент по условию | `Optional[T]` |
 `filter(predicate)` - Возвращает список элементов по условию | `List[T]` |
 `map(transform)` - Преобразует элементы в другой тип | `List[R]` |
 `remove_at_index(index)` - Удаляет элемент по индексу | `T` |
 `sort(key)` - Сортирует коллекцию | `None` |
```
