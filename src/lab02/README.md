# Лабораторная работа №2 - Коллекция объектов (Python 3.x)

## Цель работы
Научиться работать с коллекциями объектов, реализовать собственный контейнерный класс для управления группой объектов предметной области.

## Предметная область
**Оружие (Weapon)** - продолжение ЛР-1.

## Реализованная коллекция
**Inventory** - контейнер для хранения объектов класса `Weapon`.

## Функциональность

### Базовые операции (оценка 3)
-  `add(item)` - добавление объекта с проверкой типа
-  `remove(item)` - удаление объекта
-  `get_all()` - получение списка всех объектов

### Продвинутая функциональность (оценка 4)
-  `find_by_imia(name)` - поиск по имени оружия
-  `__len__()` - поддержка функции `len()`
- `__iter__()` - поддержка итерации (`for item in collection`)
- Защита от дубликатов (нельзя добавить оружие с тем же именем)

### Расширенная функциональность (оценка 5)
- `__getitem__(index)` - доступ по индексу `collection[0]`
- `remove_at_index(index)` - удаление элемента по индексу
- `sort_by_imia()` - сортировка коллекции по имени оружия
- `get_broken()` - фильтрация: получение повреждённого оружия (hardness < 0)
