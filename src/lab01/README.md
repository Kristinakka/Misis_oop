# Лабораторная работа №1
## 1. Что является сущностью?
Я создала класс Weapon(оружие) в области игровой логики, так как мне был предоставлен такой выбор
## 2. Какие у него атрибуты?
У моего класса есть атрибуты: имя, урон, редкость, твердость, эффекты, тип, количество пользователей данным оружием
Так же действия, которые можно проводить с оружием(методы): переименовать, починить, проверить атаку
## 3. Какие инварианты?
1. Твердость > 0, иначе оружие сломано, а также тип данных int 
2. Имя обязательно должно иметь символы и быть строчным типом данных 
3. Урон должен быть > 0, иначе оружие сломано. Тип данных - int
4. Редкость - собственный список 
5. Эффекты - str
6. Тип оружия - собственный список
7. Количество - int
## 4. Равенство
Оба оружия равны, если у них совпадает Тип
## 5. Состояние:
сломано
не сломано
Правила:
Нельзя наносить урон оружием, если оно сломано(твердость = 0)
