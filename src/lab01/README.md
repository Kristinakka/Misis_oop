## Course
### В работе реализован пользовательский класс Weapon, который моделирует оружие и его характеристики. Класс содержит атрибуты объекта: имя, урон, тип, эффекты, редкость, твёрдость, количество пользователей данным оружием. Также реализовано логическое состояние оружия (сломано или не сломано).

## Закрытые поля:
- _imia - имя оружия
- _damage - урон(max=100)
- _tip - тип оружия(реализуется списком)
- _effects - эффекты для оружия
- _rare - его редкость(реализуется списком)
- _hardness - твёрдость
- __kolichestvo - приватный метод, количество пользователей данным оружием

## Свойства (@property):
- status - состояние оружия
- rename - переименование оружия

## Магические методы:
- `__str__` — удобный вывод информации об оружии
- `__repr__` — техническое представление объекта
- `__eq__` — сравнение оружия по типу

## Бизнес-методы:
- repair() — починка оружия(hardness=100)
- attack() - проверка: можно ли атаковать(проверяется по hardness)
